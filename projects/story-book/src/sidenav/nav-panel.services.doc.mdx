# `Services`

## <Icon name="developer_guide"/> PanelRegistry

Le service `PanelRegistry` répertorie les panels : **panels**, **dynamic-content**, **openable**, qui ont été identifiés avec un `panelId`.

Cela permet de les retrouver depuis un autre composant en injectant ce service.
